# Pràctica — CV Web Professional (HTML5 + CSS) + Deploy  - Joan Peña Neira

## Descripción

Práctica de la asignatura "llenguatge de marques i sistemes de gestió d'informacó", donde   
se ha desarrollado un currículum vitae interactivo en formato web con las siguientes características:

- Estructura semántica HTML5
- Secciones extendidas acerca mis datos personales, adaptandolo
a un formato de CV
- Estilo CSS propio con una maquetación y estilo visual muy 
definidos (habiendo implementado tanto variables como animaciones) 
- Diseño responsive (mobile-first)
- Uso de pseudo-elements en el codigo
- Modo impresión optimizado (@media print) y enlaces con URL visible al imprimir (usando `content: attr(href)`)

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)


## Estructura principal del proyecto

La estructura del proyecto consiste en un directorio general llamado "CV_Practica_Joan_Pena", el cual contiene:

-un archivo "index.html"
-una carpeta llamada "css" donde se encuentra "style.css" y un "reset.css"
-una carpeta llamada "assets" con todos sus respectivos archivos/ documentos externos (pdf del cv y varias imagenes usadas para la pagina web)
-un archivo README.md
